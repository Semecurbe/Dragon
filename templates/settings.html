🧠 System Prompt

🔍 RAG Parameters

Controls how many chunks are retrieved and how much surrounding context is included.

1 — precise 10 — balanced 20 — broad
0 — off 2 — default 5 — max

Adds up to N chunks immediately before and after each matched chunk, giving the LLM more surrounding context at the cost of more tokens.

Vector store backend

Requires the pgvector extension on your PostgreSQL server and pip install psycopg2-binary pgvector.

⚠ Switching backends clears the active vector store — re-ingest your documents after saving.

Any model from sentence-transformers is supported. ⚠ Changing this model clears the vector database — all documents must be re-ingested.
